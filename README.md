# らぢ助 - Radio Scheduler -

-----

## 『らぢ助』とは？

Raspberry Pi にインストールを行ってスケジュール設定を行い、ラジオを自動再生させるプログラムです。
目覚まし替わりにラジオを再生させていましたが、細かい時間設定が可能なラジオが無かったため作製しました。
ラジオ以外にも音声ファイルを登録して再生させることも可能です。

### らぢ助ができること

- ラジオの任意再生
- 音声ファイルの登録 (mp3のみ)
- スケジュールに登録して任意の時間にラジオ・音声ファイルの再生
- 任意の時間にラジオを停止
